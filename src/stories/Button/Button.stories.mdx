# Buttons

Com o componente de `Button` é possível passar 3 tipos de `variants`: `contained`, `outlined`, `text`. Também pode-se passar os tipos de `color`: `primary`, `secondary` and `destruction`.

## Contained

A variant `contained` tem background-color e hover definido para cada `color`.
A prop color pode ser `primary`: green; `secondary`: gray e `destruction`: red.
Mudando-se apenas a escala de cor entre cada propriedade.

<Canvas>
  <Story name='Contained'>
    <Column alignItems='flex-start'>
      <Button color='primary' variant='contained' mb={20}>
        Primary
      </Button>
      <Button color='secondary' variant='contained' mb={20}>
        Secondary
      </Button>
      <Button color='destruction' variant='contained'>
        Destruction
      </Button>
    </Column>
  </Story>
</Canvas>

## Outlined

`Outlined` tem o mesmo padrão de Contained, porém com o background-color `transparent` quando não está aplicado o hover.

  <Story name='Outlined'>

        Primary
      </Button>
      <Button color='secondary' variant='outlined' mb={20}>
        Secondary
      </Button>
      <Button color='destruction' variant='outlined'>
        Destruction
      </Button>
    </Column>
  </Story>
</Canvas>

## Text

Por fim, a variant `Text` também segue o mesmo padrão de variantes anteriores. Nessa prop também é possível passar `color` como `primary`, `secondary` e `destruction`.
Porém, quando não aplicado estiver aplicado hover, seu background-color é 'transparent'. Mudando-se apenas para a color desejada quando estiver em hover.

        Secondary
      </Button>
      <Button color='destruction' variant='text'>
        Destruction
      </Button>
    </Column>
  </Story>
</Canvas>

# With Icons

No Design system da Atlas temos que, além das variações e colors distintas, também temos as opções de buttons com/sem ícone, na direita e/ou esquerda.
Com as props `leftIcon` e `rightIcon` é possível passar esses ícones.
